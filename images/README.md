# Images Folder

This folder contains image assets for the Errorfect project.

## Purpose

- Art pieces and visual word deconstructions
- Social media graphics
- Blog post featured images
- Product images (workbooks, courses)
- Workshop promotional materials

## Structure Suggestions

```
images/
├── art/              # Art pieces (NORMAL/PERFECT with flaming letters, etc.)
├── blog/             # Blog post featured images
├── products/         # Digital product images
├── social/           # Social media graphics
├── workshops/        # Workshop promotional images
└── branding/         # Logo, favicon, brand assets
```

## Usage

Add your images here and reference them in:
- WordPress media library (upload via admin)
- Blog posts
- Product pages
- Social media posts
- Email campaigns

## File Guidelines

- **Formats**: JPG, PNG, SVG, WebP
- **Optimization**: Compress images before committing
- **Naming**: Use descriptive, lowercase names with hyphens
  - Example: `normal-perfect-flaming-letters.jpg`
- **Sizes**: 
  - Blog featured: 1200x630px
  - Social media: 1080x1080px (square) or 1080x1350px (story)
  - Product images: 800x800px minimum
